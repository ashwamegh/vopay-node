# VoPay Node

[![npm](https://img.shields.io/npm/v/vopay-node.svg?maxAge=2592000?style=flat-square)](https://www.npmjs.com/package/vopay-node)

## About

This is an unofficial library to leverage VoPay REST API(s) for payment solutions.

## Installation

```bash
npm i vopay-node
```
or
```bash
yarn add vopay-node
```
